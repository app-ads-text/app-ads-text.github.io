Accessibility Statement

Making our applications accessible to everyone

Last Updated: May 25, 2025

Our Commitment: Test Apps Limited is committed to making our applications accessible to all users, including those with disabilities. We strive to create inclusive experiences that everyone can use comfortably and effectively.

Accessibility Features

Our applications are designed with accessibility in mind and include the following features:

Visual Accessibility

  • High Contrast Support: Our apps work seamlessly with iOS high contrast settings
  • Color-Blind Friendly: We avoid relying solely on color to convey information
  • Clear Typography: Large, readable fonts with appropriate spacing
  • Scalable Text: Full support for iOS Dynamic Type and text scaling
  • Logical Layout: Intuitive navigation and consistent interface design

Motor and Touch Accessibility

  • Large Touch Targets: Buttons and interactive elements meet minimum size requirements
  • Alternative Input: Compatible with iOS Switch Control and AssistiveTouch
  • Gesture Alternatives: Standard tap interactions don't require complex gestures

Cognitive Accessibility

  • Clear Instructions: Simple, straightforward language and guidance
  • Progress Indicators: Clear feedback on test progress and completion
  • Consistent Navigation: Standard iOS navigation patterns throughout
  • Error Prevention: Clear validation and helpful error messages

iOS Accessibility Integration

Our applications are designed to work with built-in iOS accessibility features:

Recommended iOS Accessibility Features

  • VoiceOver: Full screen reader support for all app content
  • Voice Control: Navigate and interact using voice commands
  • Switch Control: Use external switches or adaptive hardware
  • Dynamic Type: Adjust text size system-wide for better readability
  • Reduce Motion: Minimize animations and transitions
  • High Contrast: Enhance color contrast for better visibility
  • Bold Text: Make text more prominent and easier to read
  • Button Shapes: Add visual indicators to interactive elements

To access these features, go to Settings > Accessibility on your iOS device.

Ongoing Improvements

We continuously work to improve the accessibility of our applications:

  • Regular testing with iOS accessibility features
  • Following Apple's accessibility guidelines for app development
  • Incorporating user feedback to identify and address accessibility barriers
  • Staying updated with the latest accessibility standards and best practices

Feedback and Support

If you encounter any accessibility barriers while using our applications, or if you have suggestions for improvement, please let us know. Your feedback helps us make our apps better for everyone.

What to Include

When reporting an accessibility issue, please include:

  • Which app you were using
  • Your iOS version and device model
  • Which accessibility features you have enabled
  • A description of the issue you encountered
  • Any steps that might help us reproduce the problem
